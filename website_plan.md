# EBS 파이썬 교육 웹사이트 계획서

## 개요
- **프로젝트명**: EBS 이솦과 함께하는 파이썬
- **대상**: 파이썬을 공부하려는 초중고 학생
- **목적**: 교육과정 안내 및 차시별 코드 학습 제공

---

## 사이트 구조

```
홈페이지
├── 메인 페이지 (교육과정 개요)
├── Part 1. 파이썬 기초 문법 (1~10차시)
│   ├── 1차시. 파이썬의 이해와 설치
│   ├── 2차시. 수치 자료형과 변수
│   ├── ...
│   └── 10차시. 개념 복습
├── Part 2. 파이썬 라이브러리 활용 (11~18차시)
│   ├── 11차시. random 모듈
│   ├── ...
│   └── 18차시. 미니 프로젝트 - 타자 게임
├── Part 3. 웹사이트 만들기 (19~21차시)
├── Part 4. 데이터 분석 (22~24차시)
└── Part 5. 알고리즘 & 자료구조 (25~31차시)
```

---

## 페이지별 구성

### 1. 메인 페이지 (`/`)
- 교육과정 소개 (타겟, 구성, 학습 목표)
- 전체 커리큘럼 요약 (5개 Part)
- 각 Part로 빠르게 이동하는 네비게이션 카드

### 2. 차시별 상세 페이지 (`/lesson/{차시번호}`)
- **차시 제목 및 설명**
- **학습 목표**
- **코드 뷰어** (탭 구조)
  - 개념 학습 코드
  - 실습 코드 A
  - 실습 코드 B (있는 경우)
  - 도전 과제
  - 정답 (토글로 숨김 처리)
- **코드 복사 버튼**
- **구문 하이라이팅** (Python)

## 기술 스택 (권장)

### Option A: 정적 사이트 (추천 - 간단 배포)
- **프레임워크**: Next.js / Astro / Hugo
- **스타일링**: Tailwind CSS
- **코드 하이라이팅**: Prism.js / Shiki
- **배포**: GitHub Pages / Vercel / Netlify

### Option B: Streamlit (교육 내용과 연계)
- **프레임워크**: Streamlit
- **장점**: Part 3에서 배우는 내용과 동일한 기술
- **배포**: Streamlit Cloud (무료)

## 주요 기능

### 필수 기능
1. **반응형 디자인** - 모바일/태블릿/데스크톱 지원
2. **코드 뷰어** - 구문 하이라이팅, 줄 번호 표시
3. **코드 복사** - 클립보드 복사 버튼
4. **네비게이션** - Part/차시별 이동
5. **진행률 표시** - 현재 차시 위치 표시

### 선택 기능 (확장)
1. **다크 모드** - 학습 환경에 따른 테마 전환
2. **검색 기능** - 키워드로 차시/코드 검색
3. **북마크** - 학습 중인 차시 저장 (로컬스토리지)
4. **코드 실행** - Python 온라인 실행 연동 (Replit 등)
